# The Sociological Record

A visual timeline of the major sociological thinkers, organized into five eras from 1830 to the present. Built as a study reference — each thinker gets their dates, nationality, core contribution, and key terms.

![Timeline preview](sociology-timeline-infographic.png)

## What's in here

| File | Description |
|---|---|
| [`index.html`](index.html) | Interactive, scrollable timeline with a sticky era nav. Open directly in any browser — no build step needed. |
| [`sociology-timeline-infographic.png`](sociology-timeline-infographic.png) | Single-image poster version. Good for a quick-glance study sheet or printing as one page. |
| [`sociology-timeline.docx`](sociology-timeline.docx) | Word document, one era per page, ready to print and annotate. |

## Coverage

**14 thinkers across 5 eras, 1830 – present:**

1. **Founding Era** (1830s–1890s) — Comte, Marx, Spencer
2. **Classical Consolidation** (1890s–1920s) — Durkheim, Weber, Simmel
3. **Mid-Century Structuralism** (1920s–1960s) — Mead, Parsons, Merton
4. **Critical & Conflict Turn** (1960s–1980s) — Goffman, Foucault, Bourdieu
5. **Contemporary Sociology** (1980s–present) — Habermas, Giddens, Butler

## Usage

**View the interactive timeline:** open `index.html` in a browser, or enable GitHub Pages on this repo and visit the published link.

**Print the poster:** download `sociology-timeline-infographic.png` directly.

**Study/annotate:** open `sociology-timeline.docx` in Word or Google Docs.

## Notes

This is a study aid, not an exhaustive history of sociology — it favors the names and concepts most commonly covered in intro/survey courses. Contributions are intentionally condensed to one core idea per thinker; many of these scholars wrote across multiple, overlapping areas.
